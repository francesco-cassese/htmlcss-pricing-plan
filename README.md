# HTML/CSS Pricing Plan 🏷️

Esercizio di replica di un layout professionale per piani tariffari. Il focus del progetto è l'implementazione di una struttura responsiva utilizzando le proprietà avanzate di CSS e il calcolo dinamico degli spazi.

## 🎯 Obiettivo
Testare le conoscenze su:
* **Struttura semantica HTML5**: Organizzazione corretta delle sezioni.
* **Layout Responsivo**: Capacità del layout di adattarsi da mobile (1 colonna) a desktop (3 colonne).
* **Flexbox & Calcoli**: Allineamenti complessi e gestione del gap tramite la funzione `calc()`.

## 🛠️ Tecnologie Utilizzate
* **HTML5**: Per una marcatura chiara e accessibile.
* **CSS3**: Utilizzo di variabili, Flexbox.
* **Mobile-First**: Sviluppo ottimizzato partendo dalla risoluzione Mobile.

## 📱 Funzionamento Layout
* **Mobile**: I piani si incolonnano verticalmente.
* **Tablet**: I piani iniziano ad affiancarsi per ottimizzare lo spazio.
* **Desktop**: Layout a 3 colonne fisse quando lo spazio lo permette.
